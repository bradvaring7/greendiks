Policy
  • Treated only if customer specifically requests (via appointment note, yellow note, or verbal).
  • Not guaranteed → Kill-on-contact only, no long-term or residual promise.
  • Reservices → Only one reservice allowed between quarterly treatments.
  • Must set clear expectations and note communication in appointment record.
  • Script provided for explaining this limitation.
  • May treat additional pests if already servicing for a guaranteed pest.
  • Always build value by referencing target pest list in customer agreement.

Policy on Non-Covered Pests
  • Some services may affect them, but not guaranteed.
  • Must set clear expectations and document in notes.
  • If customer requests reservice for non-covered pest → Explain routine service may help, but no spot-treatment/reservice outside scheduled appointments.
  • Refer to agreement if needed.
  • Avoid selling services solely for non-covered pests, as this creates unmet expectations.
